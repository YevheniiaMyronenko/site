# Короткий сайт — Резюме та економічна безпека

Цей простий статичний сайт містить 3 сторінки:
- `Резюме` — контактні дані, освіта, навички, досягнення, досвід.
- `Економічна безпека Болгарії` — короткий висновок та графіки на основі наданих даних.
- `Загрози та протидія` — список загроз і методів протидії.

## Як розмістити на GitHub Pages
1. Створіть нове репозиторі (public або private).
2. Завантажте (upload) файли з цієї папки у корінь репозиторію.
3. У налаштуваннях репозиторію увімкніть GitHub Pages з гілки `main` (root).
4. Через хвилину сайт буде доступний за URL `https://<your-username>.github.io/<repo-name>/`.

## Структура
- `index.html`, `resume.html`, `econ.html`, `threats.html` — сторінки сайту
- `styles.css` — стилі
- `*.png` — графіки
